# Changelog

## v1.1.2

### Fixed

- Improved user interface, particularly around the protein and gene display.
- Unselecting genes now actually unselects them (not just hides them).
- Fixed up scroll area between proteins/genes and cell masks.

### Added

- Active and recent gene lists only appear after a relevant gene is selected.
- Separate tabs for COMET proteins and Xenium genes to provide more space.
- Search bar for proteins.
- Unselect and reselect genes and proteins.

## v1.1.1

### Fixed

- After selecting a gene dot size and changing cores, the set gene dot size is maintained.
- When selecting a new gene, it will instantly display the selected gene dot size.

### Added

- After selecting a gene, changing colour, and unselecting the gene, the changed colour will remain.

## v1.1.0

### Fixed

- Updated so UnumLocalia now runs on Napari 0.7.1 (up from 0.4).
- Exporting an image now maintains set gene dot size.
- Removed high and low res options. It now generates a publication-ready image that provides upscaled PNG.

### Added

- Consistent exporting for dot sizes.

## v1.0.1

### Fixed

- Notes on updating.

### Added

- Versioning.
- Changelog.
